# Gemini Advisor Queue

Role model:
- Rex: Commander
- Gemini: Strategy Advisor
- Codex: Executive Operator

Use this folder to store Gemini-facing reports, Gemini responses, and Codex action filters.

Cadence:
- Morning report: overnight execution, performance deltas, open risks.
- Evening strategy review: market direction, buyer persona, product mix, ad hypotheses.
- Decision reviews: Etsy launch mix, eBay ad settings, winner DNA expansion.

Rules:
- Do not include API keys, payment data, buyer private data, or sensitive account information.
- Save Gemini suggestions as advisory input, then Codex filters them into adopted/deferred/rejected actions.
