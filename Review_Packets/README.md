# Review Packets

Use this folder for files that Rex, Gemini, or another AI should inspect directly.

Read first:

- `OPENCLAW_GEMINI_BRIEF.md` - current copy/paste brief for Gemini. It assumes Gemini cannot see the repo/code/files.
- `REPORT_INDEX.md` - short index of what belongs here.

Optional latest supplements:

- `Latest/morning_report_latest.md`
- `Latest/gemini_review_queue_latest.md`

Rules:

- Put human-facing summaries, decision packets, contact sheets, and market diagnosis files here.
- Keep raw logs, caches, screenshots from one-off debugging, and large generated assets out of this folder.
- Prefer small Markdown or CSV files with clear dates in the filename.
- If a file is only useful to a script, keep it in `Database/` instead.

Current source-of-truth files remain:

- `START_HERE_OPENCLAW.md`
- `RECOVERY_STATE.json`
- `CURRENT_TASK.md`
- `PROGRESS_LOG.md`
- `OPERATIONS_MANUAL.md`

Important: older timestamped reports may contain obsolete assumptions. The current canonical fact is that wired LAN is working and low-bandwidth mode is lifted.
