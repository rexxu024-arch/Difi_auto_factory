## Recommendation

Best strategic lane: premium broad niche, not narrow single-theme.

The shop should not be named only around Zen, because that limits future poster/acrylic expansion. It should signal a high-end visual object studio with room for Zen, dark academia, jade, wabi-sabi, celestial, library, and collector-gift products.
